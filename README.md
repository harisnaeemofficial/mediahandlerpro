# mediahandlerpro
Media Handler Pro is a powerful .NET FFMPEG Wrapper and video processing tool which allow you to publish any format video to another format
, grab single or multiple thumbs, split or join videos, post watermark on video or image, 
retrieve useful information from media files and can process any level of custom ffmpeg commands on the fly using asp.net, .net, c# 
& vb.net applications.

For more detail, usage, documentation and sample codes visit https://www.mediasoftpro.com/media-handler-pro.html
